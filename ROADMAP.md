# 🗺️ Project Roadmap - Aplikasi Manajemen RT/RW

Dokumen ini merangkum tahapan pengembangan serta rencana fitur masa depan untuk platform **Lingkar Warga**.

---

## ✅ Tahap 1: Fondasi & Manajemen Dasar (Completed)
Fokus pada pembangunan infrastruktur inti dan fitur administrasi dasar.
- [x] Inisialisasi Database PostgreSQL & Docker.
- [x] Sistem Autentikasi JWT (Login, Register).
- [x] CRUD Data Warga (Tambah, Edit, Hapus).
- [x] Manajemen Profil Pengguna.
- [x] Setup Frontend Flutter & Integrasi API Dasar.

## 🏗️ Tahap 2: Penguatan Sistem & Alur Kerja (In Progress)
Meningkatkan efisiensi alur kerja administrasi antara Warga, RT, dan RW.
- [/] Dashboard Statistik RW (Visualisasi data warga & RT).
- [/] Sistem Verifikasi Akun & Pengajuan Perubahan Data.
- [/] Notifikasi internal untuk Admin (RT/RW).
- [ ] Optimasi performa API & Error Handling.

## 💰 Tahap 3: Modul Keuangan & Pelaporan (Planned)
Digitalisasi sistem iuran dan pembuatan laporan otomatis.
- [ ] Modul Iuran Bulanan (Tracking pembayaran warga).
- [ ] Pencatatan Pengeluaran Kas RT/RW.
- [ ] Export Laporan (PDF/Excel) untuk rapat bulanan.
- [ ] Dashboard Grafik Keuangan.

## 📱 Tahap 4: Integrasi Komunikasi (Planned)
Mempermudah koordinasi antar warga dan pengurus.
- [ ] Integrasi WhatsApp Gateway (Notifikasi tagihan & info penting).
- [ ] Sistem Pengumuman Digital (Broadcast ke seluruh warga).
- [ ] Fitur Pengaduan & Aspirasi Warga.
- [ ] Digitalisasi Surat Pengantar (E-Surat).

## 🚀 Tahap 5: Hardening & Deployment (Planned)
Persiapan untuk penggunaan skala luas dan produksi.
- [ ] Implementasi SSL & Security Hardening.
- [ ] Backup Database Otomatis.
- [ ] Testing menyeluruh (Unit Test & UI Test).
- [ ] Deployment ke Cloud Hosting / Server Mandiri.
